GitHub Pages 作成手順（英語 + 日本語）

Step Operation (English) 説明（日本語）
1 Create Repository GitHubで新しいリポジトリを作成する
2 Clone with VS Code VS Codeでリポジトリをクローンする
3 Create index.html `index.html` ファイルを作成する
4 Commit changes 変更をコミット（保存）する
5 Push to GitHub GitHubへアップロード（Push）する
6 Enable GitHub Pages GitHub Pagesの公開設定を行う
7 Open public site 公開されたページURLを開いて確認する
